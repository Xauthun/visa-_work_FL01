คำถามที่พบบ่อย (FAQ)

รวมคำถามเกี่ยวกับวีซ่า การทำงาน การอยู่ต่อ และการจดทะเบียนถิ่นที่อยู่ในประเทศไทย

ตอบ : ต้องขอวีซ่าประเภท Non-Immigrant (Type B) จากสถานทูตหรือสถานกงสุลไทยในต่างประเทศ
ดูรายละเอียด

ตอบ : ให้ติดต่อขอ Work Permit ที่กรมการจัดหางาน กระทรวงแรงงาน www.doe.go.th

ตอบ : ตรวจสอบหลักเกณฑ์ได้ที่ กดที่นี่

ตอบ : ตรวจสอบเอกสารที่ คลิกที่นี่ เลือก > การขออยู่ต่อ > กรณีมีเหตุจำเป็นทางธุรกิจ

ตอบ : สามารถยื่นได้ล่วงหน้า 30 วันก่อนครบกำหนดอนุญาต

ตอบ : ดาวน์โหลดแบบฟอร์มได้ที่ คลิกที่นี่

ตอบ : ยื่นได้ที่ บก.ตม.1 ศูนย์ราชการฯ อาคารบี ชั้น 2 ด้านทิศใต้ ดูรายละเอียดเพิ่มเติมที่ ที่นี่

ตอบ : สามารถติดต่อด่านตรวจคนเข้าเมืองประจำจังหวัดนั้นๆ หรือดูข้อมูลได้ที่ เว็บไซต์ตรวจคนเข้าเมือง

ตอบ : สามารถขออยู่ต่อได้ 7 วัน โดยใช้แบบฟอร์ม ตม.7, สำเนาหนังสือเดินทาง และค่าธรรมเนียม 1,900 บาท

ตอบ : ต้องขอวีซ่าประเภท Non-Immigrant (Type O)

ตอบ : สามารถตรวจสอบหลักเกณฑ์ได้ที่ ที่นี่

ตอบ : ได้ หากอายุ 50 ปีขึ้นไป และได้รับวีซ่า Non-Immigrant (Type O)

ตอบ : ได้ โดยต้องมีความสัมพันธ์ทั้งทางนิตินัยและพฤตินัย และใช้วีซ่า Non-Immigrant (Type O)

ตอบ : ได้ หากบุตรอายุต่ำกว่า 20 ปี ยังไม่สมรส และอาศัยอยู่ในครัวเรือนเดียวกัน

ตอบ : ไม่สามารถยื่นติดตามได้ เนื่องจากไม่ได้ระบุไว้ในหลักเกณฑ์

ตอบ : ใช้แบบฟอร์ม ตม.7 พร้อมรูปถ่าย 1 ใบ ค่าธรรมเนียม 1,900 บาท สำเนาหนังสือเดินทางของผู้ได้สิทธิและผู้ติดตาม, ใบอนุญาตทำงาน, หนังสือรับรองผู้ติดตาม และหลักฐานครอบครัวที่รับรองโดยสถานทูตและกรมการกงสุล

ตอบ : ยื่นที่ ตม. เดิมที่เคยยื่นคำร้องขออยู่ต่อของผู้มี work permit

ตอบ : สามารถขออยู่ต่อได้อีก 7 วัน (นับจากวันที่พ้นหน้าที่) โดยใช้ ตม.7 สำเนาหนังสือเดินทาง รูปถ่าย 1 ใบ และค่าธรรมเนียม 1,900 บาท

ตอบ : นายจ้างต้องแจ้งยกเลิกวีซ่าที่ ตม. โดยใช้หนังสือแจ้งพ้นหน้าที่, หนังสือรับรองการจดทะเบียนบริษัท, สำเนาบัตรประชาชนกรรมการ, และหนังสือเดินทางของคนต่างด้าว

ตอบ : ต้องแจ้งที่พักอาศัยทุก 90 วัน โดยใช้แบบฟอร์ม ตม.47

ตอบ : แจ้งเข้าพักอาศัยต่อ ตม. ภายใน 24 ชั่วโมง หรือส่งทางไปรษณีย์ลงทะเบียน ใช้แบบฟอร์ม ตม.30

ตอบ : หนังสือเดินทางฉบับจริง, แบบฟอร์ม ตม.47 ที่กรอกและลงชื่อแล้ว, ใบรับแจ้งครั้งสุดท้าย (ถ้ามี)

ตอบ : มารายงานตัวเอง โดนปรับ 2,000 บาท / หากเจ้าหน้าที่ตรวจพบ โดน 4,000 บาท และ 200 บาทต่อวัน

ตอบ : ได้ โดยนำหนังสือเดินทางและแบบฟอร์ม ตม.47 ที่ลงนามไว้แล้วมาแจ้งแทน

ตอบ : แจ้งได้ที่ ตม. ในเขตพื้นที่ที่พักอาศัยอยู่ หรือผ่านทางไปรษณีย์ลงทะเบียน

ตอบ : กองกำกับการ 2 กองตรวจคนเข้าเมือง 1 ศูนย์ราชการฯ อาคารบี ชั้น 2 ด้านทิศใต้ ถนนแจ้งวัฒนะ

ตอบ : ส่งเอกสารตามที่กำหนด รวมถึงซองติดแสตมป์ 10 บาท สำหรับส่งใบนัดครั้งต่อไป ให้ ตม. ที่พักอาศัยอยู่

ตอบ : แจ้งที่สำนักงานตรวจคนเข้าเมือง หรือสถานีตำรวจในท้องที่

ตอบ : รับได้ที่สำนักงานตรวจคนเข้าเมืองทั่วประเทศ หรือดาวน์โหลดที่ www.immigration.go.th

ตอบ : ต้องแจ้งเมื่อครบ 90 วันนับจากวันที่เดินทางกลับเข้ามาในประเทศไทยครั้งสุดท้าย

ตอบ : ไม่ถือว่าเป็นการแจ้งที่พักอาศัย 90 วัน ต้องใช้แบบฟอร์ม ตม.47 เท่านั้น

ตอบ : แจ้งความที่สถานีตำรวจ ติดต่อสถานทูตขอเล่มใหม่ แล้วนำใบแจ้งความและเล่มใหม่ไปแจ้ง ตม.

ตอบ : ต้องมาติดต่อด้วยตนเองเท่านั้น

ตอบ : ไม่ได้ จะได้รับอนุญาตให้อยู่ต่อเพียง 7 วัน เพื่อเดินทางกลับประเทศ

ตอบ : ได้ โดยต้องมีวีซ่าประเภท Non-O และแสดงหลักฐานความสัมพันธ์ รายได้เฉลี่ยไม่ต่ำกว่า 40,000 บาท/เดือน หรือเงินฝาก 400,000 บาท

ตอบ : ต้องได้รับวีซ่า Non-O แสดงหลักฐานความสัมพันธ์, บัตรประชาชนสามี, รูปถ่ายครอบครัว, แผนที่บ้าน

ตอบ : ที่ บก.ตม.1 ศูนย์ราชการฯ อาคารบี แจ้งวัฒนะ หรือ ตม.ในพื้นที่ที่พักอาศัยอยู่

ตอบ : ได้ ต้องแสดงหลักฐานความสัมพันธ์และมีรายได้ 40,000 บาท/เดือน หรือเงินฝาก 400,000 บาท

ตอบ : ได้ หากลงทุนไม่น้อยกว่า 10 ล้านบาท เช่น ซื้อคอนโด, ฝากเงิน หรือซื้อพันธบัตรรัฐบาล

ตอบ : ต้องอายุไม่เกิน 20 ปี และยังไม่ได้สมรส อยู่ในครัวเรือนเดียวกัน

ตอบ : ต้องอายุ 50 ปีขึ้นไป มีเงินฝาก 800,000 บาท หรือรายได้เดือนละ 65,000 บาท หรือรวมกันปีละ 800,000 บาท

ตอบ : ได้ รวมเงินฝากกับรายได้ให้ครบ 800,000 บาทต่อปี

ตอบ : ไม่ได้ ต้องเป็นบัญชีของผู้ยื่นเท่านั้น

ตอบ : ได้ ต้องมีหนังสือรับรองจากแพทย์ที่ระบุอาการและระยะเวลาการรักษา

ตอบ : ได้ หากมีหลักฐานยืนยันว่าเป็นผู้เกี่ยวข้องในคดี เช่น ผู้เสียหาย โจทก์ จำเลย หรือพยาน

ตอบ : ได้ โดยต้องแสดงหลักฐานว่าเคยมีสัญชาติไทย เช่น บัตรประชาชน ทะเบียนบ้าน สูติบัตร

ตอบ : ยื่นขออยู่ต่อด้วยเหตุพิสูจน์สัญชาติ โดยได้รับการรับรองจากหน่วยงานที่เกี่ยวข้อง

ตอบ : ยื่นขออยู่ต่อ โดยต้องมีหนังสือรับรองจากองค์กรที่เกี่ยวข้อง

ตอบ : ไม่จำเป็น สามารถมอบอำนาจให้ผู้อื่นมาดำเนินการแทนได้

ตอบ : ต้องให้สถานทูตเจ้าของเอกสารรับรองการแปล และนำไปกรมการกงสุล กระทรวงการต่างประเทศ รับรองอีกครั้ง
